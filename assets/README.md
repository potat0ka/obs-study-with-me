# Assets

This folder is reserved for images, icons, screenshots, or other package assets related to the OBS Study With Me Pomodoro tool.

## Suggested assets to add

| File | Purpose |
|---|---|
| `logo.png` / `logo.svg` | Project logo for README and releases |
| `screenshot-focus.png` | Screenshot of the Focus scene in OBS |
| `screenshot-break.png` | Screenshot of the Break scene in OBS |
| `screenshot-dock.png` | Screenshot of the browser dock UI |
| `favicon.ico` | Browser favicon for the dock UI |

## Usage
Reference assets from `README.md` or docs using relative paths:
```markdown
![screenshot](assets/screenshot-focus.png)
```
