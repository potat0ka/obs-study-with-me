# OBS Study With Me Manager 🍅🎓

An advanced Pomodoro and "Study With Me" stream manager for OBS Studio, co-created with Agentic AI.  
This Lua script completely automates your study streams so you can focus on working, not on clicking buttons in OBS.

## 📦 Package Overview
This repository is organized as a plugin-style distribution package:
- `lua/` — packaged OBS Lua script (`pomorodo.lua`)
- `dock/` — browser dock UI for remote control via OBS WebSocket
- `docs/` — installation and quickstart guides
- `install/` — installer helper notes
- `release/` — packaging and release workflow
- `assets/` — icons, screenshots, or resource assets

For the fastest setup, see `docs/QUICKSTART.md`.

## ✨ Features

- **🪄 1-Click Auto-Setup**: Automatically generates **FOUR** scenes: `Study With Me - Prep`, `Study With Me - Focus`, `Study With Me - Short Break`, and `Study With Me - Long Break`. It automatically adds and positions all necessary text and media sources in all of them.
- **⏱ Prep / Countdown Phase**: An optional prep scene plays before each focus session with a live countdown (configurable seconds), so your viewers know focus is starting.
- **🎬 Auto-Scene Switching**: Automatically switches between your Prep, Focus, Short Break, and Long Break scenes based on the active timer.
- **🛑 Stream Auto-Stop**: Going to bed? The script can automatically end your live stream when your final Long Break is over.
- **📊 Daily Goal Tracker**: Keeps track of how many Pomodoro sessions you have completed during your stream.
- **🔔 Toast Notifications**: On-screen pop-up notifications when segments change (e.g. "Focus started", "Short break started").
- **📋 Status Panel Source**: A detailed multi-line text source showing current mode, time remaining, session progress, subject, next segment, and daily goal — great for your stream overlay or dashboard.
- **💬 Chat Control**: Optional Twitch/chat bot integration — viewers (or mods only) can control the timer with `!start`, `!pause`, `!resume`, `!toggle`, `!skip`, `!reset`, `!stop`, `!focus`, `!break`, `!longbreak`.
- **🔊 Reliable Audio Routing**: The script creates ONE unified `Pomodoro Alert` audio source in your mixer that dynamically plays your sound files. No phantom audio loading, and full control over your volume slider!
- **🎵 Per-Scene Background Music**: Assign separate looping BGM audio files for Focus, Short Break, and Long Break scenes.
- **🕒 Local Stream Clock**: Displays your real-time local clock on stream with a configurable format.
- **🎨 Customizable Colors**: Set distinct timer text colors for each mode (Focus, Short Break, Long Break, Prep, Paused, Stopped).
- **⌨️ Full Hotkey Support**: Bind keyboard shortcuts or Stream Deck buttons for every timer action.
- **🌐 Browser Dock Control**: A companion web dock lets you control the timer via OBS WebSocket from any browser or Stream Deck.

## 📦 Installation & Setup

1. Open OBS Studio.
2. Go to **Tools → Scripts**.
3. Click the **+** button and load `lua/pomorodo.lua`.
4. Select the script in the menu. At the very top of the properties, click the **🪄 Auto-Create Scene Setup (Recommended)** button. (This will instantly create your scenes and all your text sources!)
5. Click **Defaults** at the bottom of the script window to auto-link the script to your newly created scenes and sources.
6. Under the **Sounds** section in the script, click **Browse** to select your `.mp3` or `.wav` files for the Focus, Break, and Warning alerts.

## 🔊 Fixing Audio Monitoring (Important!)

Because this script uses standard OBS Media Sources, you need to tell OBS to output the sound to your headphones!
1. Check your OBS **Audio Mixer** — you will see a new source called `Pomodoro Alert` (it was generated by the Auto-Setup button).
2. Click the **Gear Icon ⚙️** next to it, and open **Advanced Audio Properties**.
3. Find `Pomodoro Alert` and change **Audio Monitoring** to **"Monitor and Output"**.

## ⌨️ Hotkeys

The following hotkeys are registered by the script and can be bound in OBS **Settings → Hotkeys**:

| Hotkey ID | Action |
|---|---|
| `pomo_start` | Start the timer |
| `pomo_pause` | Pause the timer |
| `pomo_resume` | Resume from pause |
| `pomo_toggle` | Toggle start/pause/resume |
| `pomo_skip` | Skip to the next segment |
| `pomo_reset` | Reset the timer |
| `pomo_stop` | Stop the timer entirely |
| `pomo_focus` | Jump directly to Focus mode |
| `pomo_short_break` | Jump directly to Short Break |
| `pomo_long_break` | Jump directly to Long Break |

These same hotkey IDs are used by the browser dock and OBS WebSocket integration.

## 🌐 Browser Dock

Use `dock/index.html` as a Custom Browser Dock in OBS (`View → Docks → Custom Browser Docks`). It connects to OBS WebSocket and gives you one-click control over all timer actions.

## 📺 Auto-Created Sources

When you click **Auto-Create Scene Setup**, the following sources are created:

**Scenes:**
- `Study With Me - Prep`
- `Study With Me - Focus`
- `Study With Me - Short Break`
- `Study With Me - Long Break`

**Text Sources (added to all scenes):**
- `Pomodoro Timer` — the live countdown display
- `Pomodoro Status` — current mode message
- `Pomodoro Clock` — real-time local clock
- `Pomodoro Goal` — daily session goal tracker
- `Pomodoro Subject` — current study subject/task
- `Pomodoro Panel` — detailed multi-line status panel
- `Pomodoro Toast` — pop-up transition notifications

**Audio Sources:**
- `Pomodoro Alert` — shared non-looping sound alert source
- `Pomodoro BGM - Focus` — looping BGM for Focus scene
- `Pomodoro BGM - Short Break` — looping BGM for Short Break scene
- `Pomodoro BGM - Long Break` — looping BGM for Long Break scene

## 💡 Customization
Once you hit the Auto-Setup button, you can go into your scenes and freely move, resize, and change the fonts of the generated Text Sources. You can also add your webcam, background images, or custom overlays! The script will simply update the text values in the background.
